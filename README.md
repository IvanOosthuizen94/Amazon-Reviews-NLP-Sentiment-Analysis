# Amazon-Reviews-NLP-Sentiment-Analysis
A Natural Language Processing (NLP) project applying sentiment analysis to Amazon consumer reviews. Built with Python, spaCy, and TextBlob, the model processes and evaluates text data to calculate polarity scores, classifying customer feedback as positive, negative, or neutral to uncover actionable business insights.

1. Navigate to the directory  
2. Download the Dataset: Download the CSV from the Kaggle link below and place it in the root directory of this repository.  https://www.kaggle.com/datasets/datafiniti/consumer-reviews-of-amazon-products
3. Install Dependencies (Navigate to the project directory and install the required dependencies using pip:
pip install -r requirements.txt)   
4. Launch Jupyter Notebook  
5. Open the Notebook: Open sentiment_analysis.ipynb and run the cells to execute the sentiment analysis.  

### Project Structure  
1. sentiment_analysis.ipynb: The main notebook containing the NLP models, data preprocessing, and polarity measurement logic.  
2. sentiment_analysis_report.pdf: The final analytical report detailing the methodology, preprocessing steps, and evaluation of the sentiment results.
